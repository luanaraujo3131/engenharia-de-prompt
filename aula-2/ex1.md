# PROMPT
## Explique o que são redes neurais
Redes neurais são modelos de inteligência artificial inspirados no funcionamento do cérebro humano. Elas são muito usadas dentro do campo de Aprendizado de Máquina para reconhecer padrões, fazer previsões e tomar decisões.

🧠 Ideia básica

Assim como o cérebro tem neurônios conectados, uma rede neural é formada por “neurônios artificiais” organizados em camadas:

Camada de entrada → recebe os dados (ex: imagem, texto, números)
Camadas ocultas → processam as informações
Camada de saída → entrega o resultado (ex: “é um gato” ou “é um cachorro”)
⚙️ Como funciona na prática

Cada conexão entre neurônios tem um “peso” (um número).
A rede faz cálculos com esses pesos para gerar um resultado.

Depois, ela compara o resultado com o esperado e ajusta os pesos — isso é chamado de aprendizado.

Esse processo usa conceitos matemáticos como:

Gradiente Descendente → método para ajustar os pesos
Função de Ativação → decide se um neurônio “ativa” ou não
📊 Exemplo simples

Imagine que você quer ensinar uma rede a identificar se um e-mail é spam:

Você mostra vários exemplos de e-mails
A rede analisa palavras, padrões, etc.
Ela aprende a diferenciar spam de não-spam
Depois consegue classificar novos e-mails sozinha
🚀 Onde são usadas

Redes neurais estão por toda parte:

Reconhecimento facial (ex: desbloqueio do celular)
Assistentes virtuais (como ChatGPT)
Recomendação de filmes e músicas
Carros autônomos
Diagnóstico médico
🧩 Tipos comuns
Rede Neural Simples (Perceptron)
Redes Neurais Convolucionais (CNNs) → imagens
Redes Recorrentes (RNNs) → textos e sequências
Transformers → usados em modelos modernos de linguagem

## Suponhamos que você seja um professor especialiado em inteligencia artifical, explique redes neurais para um aluno iniciando no tema. Faça uma explicação simples, com exemplo e limitado a 3 paragrafos

Redes neurais são modelos de inteligência artificial inspirados no cérebro humano, usados dentro do Aprendizado de Máquina. Elas são formadas por “neurônios artificiais” organizados em camadas: a de entrada (recebe os dados), algumas camadas internas (processam) e a de saída (dão o resultado). Cada conexão tem um peso, que indica a importância daquela informação no resultado final.

O aprendizado acontece quando a rede ajusta esses pesos com base nos erros que comete. Ela faz uma previsão, compara com a resposta correta e vai se corrigindo aos poucos usando técnicas como o Gradiente Descendente. Com o tempo e muitos exemplos, ela melhora e passa a acertar cada vez mais.

Um exemplo simples: imagine ensinar a rede a identificar se uma foto é de um gato ou cachorro. Você mostra milhares de imagens já classificadas. No começo ela erra bastante, mas conforme aprende padrões (orelhas, focinho, formato), ela começa a acertar. Depois de treinada, consegue olhar uma nova imagem e dizer sozinha se é gato ou cachorro.
