# Calculadora Simples

No primeiro arquivo(calculator), foi criado um mini protopipo de calculadora: uma versão procedural e sequencial usando funções e estruturas condicionais

Posterormente foi pedido uma versão refatorada a IA, que teve como resultado um código menos verboso e redundante.

⚙️ Funcionalidades

Ambas as versões têm:

Adição
Subtração
Multiplicação
Divisão (com verificação de divisão por zero)
Constantes:
Pi (π)
Número de Euler (e)

A versão refatorada tem como características a utiliza uma classe Calculator, operações armazenadas em um dicionário, execução dinâmica das funções, melhor validação de entrada, uso do módulo padrão math
